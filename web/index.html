Installation

  1. Fork le repo tele7sport
  2. Ajoutez votre favorites.json à la racine
  3. Dans Settings → Secrets → Actions, ajoutez :
    • EMAIL_USER votre adresse Gmail
    • EMAIL_PASSWORD votre mot de passe d'application
    • EMAIL_TO adresse destinataire
  4. Activez GitHub Actions
  5. Newsletter chaque lundi matin
