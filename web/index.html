🚀 Installation rapide

  1. Forkez le repo tele7sport
  2. Collez votre favorites.json à la racine du projet
  3. Allez dans Settings → Secrets → Actions et ajoutez :
    • EMAIL_USER — votre adresse Gmail
    • EMAIL_PASSWORD — votre mot de passe d'application
    • EMAIL_TO — adresse destinataire
    • FAVORITES_JSON — (optionnel) le contenu JSON directement
  4. Activez GitHub Actions dans l'onglet Actions
  5. La newsletter arrive chaque lundi matin ! 📬
